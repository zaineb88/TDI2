Instructions

1

Dans chaque tableau, mettre la dernière ligne en italique.

2

Dans le corps des tableaux, attribuer à la première cellule de chaque ligne le style suivant :

	color: #555555;
	font-weight: bold;
	text-align: left;

Utiliser pour cela le format JSON.

3

Attribuer à la ligne ayant l'identifiant reg24ref la couleur de fond #FBEC88

4

Pour aller plus loin

Attribuer aux lignes paires des tableaux (dans le tbody) la couleur de fond #e8e8e8, sauf si elles ont l'identifiant #reg24ref.

Astuces

Le nom des paramètres dans le format JSON supporte les majuscules, mais ne supporte pas les tirets. Par exemple font-size devient fontSize

Le sélecteur :not(...) capture les noeuds qui ne vérifient pas une condition.
