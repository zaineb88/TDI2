Instructions

1

Masquer les items du menu déroulant.

Le menu se déplie vers le bas quand on clique sur le lien de classe menu-tete. Il se replie vers le haut quand on clique à nouveau le lien, et ainsi de suite.

2

Cacher les tableaux.

Quand on clique sur un lien du menu déroulant, le tableau lié apparaît avec un effet de votre choix.

Le cas échéant, les autres tableaux disparaissent.

3

Créer un effet quand on survole les items du menu déroulant (par exemple, modifier la propriété padding-left du style des liens).

4

Désactiver les animations, déclencher l'évènement click sur le premier item du menu des onglets. Réactiver les animations.
