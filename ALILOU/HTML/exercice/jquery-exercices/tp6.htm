Ajax

Emplacement du tableau : <div id="tableau"></div>

Exemple

Instructions

1

Affichage du tableau correspondant à l'option sélectionnée quand on clique sur le bouton «voir les départements».

Les addresses des fichiers des tableaux sont dans les attributs value des options.

Le bouton cliquable se retrouve grâce à l'identifiant valide et la liste de sélection avec l'identifiant regions.

2

Recréer la liste de sélection à partir du fichier xml/regions.xml. Les options générées à partir du xml auront comme attribut value le code de la région (regid).

3

Charger le fichier xml/departements.xml. Quand on clique sur le bouton «voir les départements», un tableau est généré avec uniquement les départements de la région sélectionnée.
